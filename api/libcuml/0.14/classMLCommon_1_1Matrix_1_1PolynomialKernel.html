Evaluate kernel matrix using polynomial kernel.

output_[i + k*n1] = (gain*<x1_i, x2_k> + offset)^exponent, where x1_i is the i-th vector from the x1 set, and x2_k is k-th vector in the x2 set, and < , > denotes dot product.

[in] n1 number vectors in x1
[in] n_cols number of features in x1 and x2
[in] x2 device array of vectors in column major format,
[in] n2 number vectors in x2 size [n2*n_cols]
[out] out device buffer to store the Gram matrix in column major format, size [n1*n2]
