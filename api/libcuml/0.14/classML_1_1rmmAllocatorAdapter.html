Detailed Description

Implemententation of ML::deviceAllocator using the RAPIDS Memory Manager (RMM) for allocations.

rmmAllocatorAdapter does not initialize RMM. If RMM is not initialized on construction of rmmAllocatorAdapter allocations fall back to cudaMalloc.
