Detailed Description

template<typename GemmTraits_, typename BaseClass = cutlass::gemm::Gemm<GemmTraits_>>
struct MLCommon::LinAlg::CustomGemm< GemmTraits_, BaseClass >

main Gemm class to launch the kernel. It is customized to accept a device lambda directly. This is done this way since cutlass currently doesn't expose such an interface.
