Detailed Description

template<typename TypeV, typename TypeK, int N, bool Greater>
struct MLCommon::Selection::KVArray< TypeV, TypeK, N, Greater >

Struct to abstract an array of key-val pairs. It is assumed to be strided across warp. Meaning, this array is assumed to be actually of length N*32, in row-major order. In other words, all of arr[0] across all threads will come first, followed by arr[1] and so on.
