Kernel to compute a batched SpMV: alpha*A*x + beta*y (where A is a sparse matrix, x and y dense vectors)

Note
One thread per batch (this is intended for very large batches) Rows don't have the same number of non-zero elements, so an approach to parallelize on the rows would lead to divergence
