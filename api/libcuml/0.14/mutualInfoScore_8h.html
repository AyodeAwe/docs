Detailed Description

The Mutual Information is a measure of the similarity between two labels of the same data.This metric is independent of the absolute values of the labels: a permutation of the class or cluster label values won't change the score value in any way. This metric is furthermore symmetric.This can be useful to measure the agreement of two independent label assignments strategies on the same dataset when the real ground truth is not known.
