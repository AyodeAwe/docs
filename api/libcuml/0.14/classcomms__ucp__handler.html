Asynchronously send data to the given endpoint using the given tag.

If the request didn't fail, but it's not OK, it is in flight. Expect the handler to be invoked

If the request is OK, it's already been completed and we don't need to wait on it. The request will be a nullptr, however, so we need to create a new request and set it to completed to make the "waitall()" function work properly.
