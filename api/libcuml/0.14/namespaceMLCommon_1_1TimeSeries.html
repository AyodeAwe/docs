Host Function to batchwise transform/inverse transform the moving average coefficients/autoregressive coefficients according to "jone's (1980)" transformation.

Parameters
params 2D array where each row represents the transformed MA coefficients of a transformed model
batchSize the number of models in a batch (number of rows in params)
parameter the number of coefficients per model (basically number of columns in params)
newParams the inverse transformed params (output)
isAr set to true if the params to be transformed are Autoregressive params, false if params are of type MA
isInv set to true if the transformation is an inverse type transformation, false if regular transform
allocator object that takes care of temporary device memory allocation of type std::shared_ptr<MLCommon::deviceAllocator>
stream the cudaStream object

◆ stationarity()

template<typename DataT , typename IdxT >
int MLCommon::TimeSeries::stationarity ( const DataT * y_d ,
int * d ,
IdxT n_batches ,
IdxT n_samples ,
std::shared_ptr< MLCommon::deviceAllocator > allocator ,
cudaStream_t stream ,
DataT pval_threshold = 0.05

Compute recommended trend parameter (d=0 or 1) for a batched series.

This function operates a stationarity test on the given series and for the series that fails the test, differenciates them and runs the test again on the first difference.

Note
The data is a column-major matrix where the series are columns. The output is an array of size n_batches.
Template Parameters
DataT Scalar type of the data (float or double)
IdxT Integer type of the indices
Parameters
[in] y_d Input data
[out] d Integer array to store the trends
[in] n_batches Number of batches
[in] n_samples Number of samples
[in] allocator cuML device memory allocator
[in] stream CUDA stream
[in] pval_threshold P-value threshold above which a series is considered stationary
Returns
An integer to track if some series failed the test
Return values
-1 Some series failed the test
0 All series passed the test for d=0
1 Some series passed for d=0, the others for d=1
