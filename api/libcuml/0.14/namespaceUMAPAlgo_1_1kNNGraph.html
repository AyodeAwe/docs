This function performs a k-nearest neighbors against the input algorithm using the specified knn algorithm. Only algorithm supported at the moment is brute force knn primitive.

Template Parameters
T Type of input, query, and dist matrices. Usually float
Parameters
X Matrix to query (size n x d) in row-major format
n Number of rows in X
query Search matrix in row-major format
q_n Number of rows in query matrix
d Number of columns in X and query matrices
knn_indices Return indices matrix (size n*k)
knn_dists Return dists matrix (size n*k)
n_neighbors Number of closest neighbors, k, to query
params Instance of UMAPParam settings
d_alloc device allocator
stream cuda stream to use
algo Algorithm to use. Currently only brute force is supported

Initial algo uses FAISS indices
