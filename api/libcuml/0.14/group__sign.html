performs fit and transform operations for the tsvd. Generates transformed data, eigenvectors, explained vars, singular vals, etc.

Parameters
[in] handle the internal cuml handle object
[in] input the data is fitted to PCA. Size n_rows x n_cols. The size of the data is indicated in prms.
[out] trans_input the transformed data. Size n_rows * n_components.
[out] components the principal components of the input data. Size n_cols * n_components.
[out] explained_var explained variances (eigenvalues) of the principal components. Size n_components * 1.
[out] explained_var_ratio the ratio of the explained variance and total variance. Size n_components * 1.
[out] singular_vals singular values of the data. Size n_components * 1
[in] prms data structure that includes all the parameters from input size to algorithm.
