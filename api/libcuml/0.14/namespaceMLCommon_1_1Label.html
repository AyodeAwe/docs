Assign one versus rest labels.

The output labels will have values +/-1: y_out = (y == y_unique[idx]) ? +1 : -1;

The output type currently is set to math_t, but for SVM in principle we are free to choose other type for y_out (it should represent +/-1, and it is used in floating point arithmetics).

Parameters
[in] y device array if input labels, size [n]
[in] n number of labels
[in] y_unique device array of unique labels, size [n_classes]
[in] n_classes number of unique labels
[out] y_out device array of output labels
[in] idx index of unique label that should be labeled as 1
[in] stream cuda stream

Maps an input array containing a series of numbers into a new array where numbers have been mapped to a monotonically increasing set of labels. This can be useful in machine learning algorithms, for instance, where a given set of labels is not taken from a monotonically increasing set. This can happen if they are filtered or if only a subset of the total labels are used in a dataset. This is also useful in graph algorithms where a set of vertices need to be labeled in a monotonically increasing order.

Template Parameters
Type the numeric type of the input and output arrays
Lambda the type of an optional filter function, which determines which items in the array to map.
Parameters
out output label array with labels assigned monotonically
in input label array
N number of elements in the input array
stream cuda stream to use
