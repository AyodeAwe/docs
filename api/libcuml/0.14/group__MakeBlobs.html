Detailed Description

GPU-equivalent of sklearn.datasets.make_blobs as documented here: https://scikit-learn.org/stable/modules/generated/sklearn.datasets.make_blobs.html.

Parameters
outthe generated data on device (dim = n_rows x n_cols) in row-major layout
labelslabels for the generated data on device (dim = n_rows x 1)
n_rowsnumber of rows in the generated data
n_colsnumber of columns in the generated data
n_clustersnumber of clusters (or classes) to generate
centerscenters of each of the cluster, pass a nullptr if you need this also to be generated randomly (dim = n_clusters x n_cols). This is expected to be on device
cluster_stdstandard deviation of each of the cluster center, pass a nullptr if you need this to be read from 'cluster_std_scalar'. (dim = n_clusters x 1) This is expected to be on device
cluster_std_scalarif 'cluster_std' is nullptr, then use this as the standard deviation across all dimensions.
shuffleshuffle the generated dataset and labels
center_box_minmin value of the box from which to pick the cluster centers. Useful only if 'centers' is nullptr
center_box_maxmax value of the box from which to pick the cluster centers. Useful only if 'centers' is nullptr
seedseed for the RNG
