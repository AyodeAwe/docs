Fast Dimensionality reduction via TSNE using the Barnes Hut O(NlogN) approximation.

Parameters
[in] VAL The values in the attractive forces COO matrix.
[in] COL The column indices in the attractive forces COO matrix.
[in] ROW The row indices in the attractive forces COO matrix.
[in] NNZ The number of non zeros in the attractive forces COO matrix.
[in] handle The GPU handle.
[out] Y The final embedding. Will overwrite this internally.
[in] n Number of rows in data X.
[in] theta repulsion threshold
[in] epssq A tiny jitter to promote numerical stability.
[in] early_exaggeration How much early pressure you want the clusters in TSNE to spread out more.
[in] exaggeration_iter How many iterations you want the early pressure to run for.
[in] min_gain Rounds up small gradient updates.
[in] pre_learning_rate The learning rate during the exaggeration phase.
[in] post_learning_rate The learning rate after the exaggeration phase.
[in] max_iter The maximum number of iterations TSNE should run for.
[in] min_grad_norm The smallest gradient norm TSNE should terminate on.
[in] pre_momentum The momentum used during the exaggeration phase.
[in] post_momentum The momentum used after the exaggeration phase.
[in] random_state Set this to -1 for pure random intializations or >= 0 for reproducible outputs.
