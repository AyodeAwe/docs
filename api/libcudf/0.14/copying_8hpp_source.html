505 /*
506 * @brief Returns a new column, where each element is selected from either @p lhs or
507 * @p rhs based on the value of the corresponding element in @p boolean_mask
508 *
509 * Selects each element i in the output column from either @p rhs or @p lhs using the following rule:
510 * output[i] = (boolean_mask.valid(i) and boolean_mask[i]) ? lhs : rhs[i]
511 *
512 * @throws cudf::logic_error if lhs and rhs are not of the same type
513 * @throws cudf::logic_error if boolean mask is not of type bool8
514 * @throws cudf::logic_error if boolean mask is not of the same length as rhs
515 * @param[in] left-hand scalar
516 * @param[in] right-hand column_view
517 * @param[in] column of `BOOL8` representing "left (true) / right (false)" boolean for each element and
518 * null element represents false.
519 * @param[in] mr resource for allocating device memory
520 *
521 * @returns new column with the selected elements
522 */
