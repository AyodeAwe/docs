Detailed Description

Find index of a given key within a dictionary's keys column.

The index is the position within the keys column where the given key (scalar) is found. The keys column is sorted and unique so only one value is expected. The result is an integer scalar identifing the index value. If the key is not found, the resulting scalar has is_valid()=false .
