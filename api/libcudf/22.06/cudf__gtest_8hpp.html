Detailed Description

Work around for GTests emulation of variadic templates in. 

::Testing::Types
Note
Instead of including gtest/gtest.h, all libcudf test files should include cudf_gtest.hpp instead.

Removes the 50 type limit in a type-parameterized test list.

Uses macros to rename GTests's emulated variadic template types and then redefines them properly.
