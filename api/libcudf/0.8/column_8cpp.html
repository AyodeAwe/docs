◆ gdf_column_concat()

gdf_error gdf_column_concat ( gdf_column * output ,
gdf_column * columns_to_concat [],
int num_columns
)

Concatenates multiple gdf_columns into a single, contiguous column, including the validity bitmasks.

Note that input columns with nullptr validity masks are treated as if all elements are valid.

Parameters
[out] output_column A column whose buffers are already allocated that will contain the concatenation of the input columns data and validity bitmasks
[in] columns_to_concat[] The columns to concatenate
[in] num_columns The number of columns to concatenate
Returns
gdf_error GDF_SUCCESS upon completion; GDF_DATASET_EMPTY if any data pointer is NULL, GDF_COLUMN_SIZE_MISMATCH if the output column size != the total size of the input columns; GDF_DTYPE_MISMATCH if the input columns have different datatypes.
