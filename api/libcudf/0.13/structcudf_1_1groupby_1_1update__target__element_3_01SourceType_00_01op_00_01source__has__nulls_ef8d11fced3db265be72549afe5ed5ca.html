Performs in-place update of a target element via a binary operation with a source element.

------------------------------------------------------------------------—*

Note
It is assumed the source element is not NULL, i.e., a NULL source element should be detected before calling this function.
If source_has_nulls==true , it is assumed that target is nullable

If the target element is NULL, it is assumed that the target element was initialized with the identity of the aggregation operation. The target is updated with the result of the aggregation with the source element, and the target column's bitmask is updated to indicate the target element is no longer NULL.
