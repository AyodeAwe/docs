Detailed Description

template<typename T>
class hostdevice_vector< T >

A helper class that wraps fixed-length device memory for the GPU, and a mirror host pinned memory for the CPU.

This abstraction allocates a specified fixed chunk of device memory that can initialized upfront, or gradually initialized as required. The host-side memory can be used to manipulate data on the CPU before and after operating on the same data on the GPU.
