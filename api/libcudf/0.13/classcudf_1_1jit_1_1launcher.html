C'tor of the launcher class.

Method to generate vector containing all template types for a JIT kernel. This vector is used to get the compiled kernel for one set of types and set it as the kernel to launch using this launcher.

Parameters
hash The hash to be used as the key for caching
cuda_code The CUDA code that contains the kernel to be launched
header_names Strings of header_names or strings that contain content of the header files
compiler_flags Strings of compiler flags
file_callback a function that returns header file contents given header file names.
stream The non-owned stream to use for execution
